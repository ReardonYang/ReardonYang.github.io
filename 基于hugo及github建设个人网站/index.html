基于hugo及github建设个人网站

2023-02-01

以下教程适用于Windows系统下，基于hugo及github，建设个人网站。

  • 第一步，安装hugo 安装最新版hugo，https://github.com/gohugoio/hugo/releases

  • 第二步，安装Visual Studio Code

  • 第三步，安装Git，https://git-scm.com/download/win

git config –global user.name “Reardon” git config –global user.email “reardonyang@gmail.com”

  • 常用命令：

cd reardonyang.com

git init

git remote add origin https://github.com/ReardonYang/ReardonYang.github.io.git

git pull origin master

git add .

git commit -m “备注”

git push -u origin master
