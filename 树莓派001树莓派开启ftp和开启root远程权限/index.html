树莓派001：树莓派开启ftp和开启root远程权限

2021-10-16

  • 开启FTP，安装vsftpd 指令：sudo apt-get install vsftpd

但是这样开启之后，ftp工具是可以连接了，但是只能从树莓派上获取文件，却无法上传

需要修改树莓派的配置 sudo vim /etc/vsftpd.conf

找到以下选项修改 anonymous_enable=NO local_enable=YES
write_enable=YES local_umask=022

然后重启ftp服务

sudo service vsftpd restart

启用root 1：修改root密码,默认是没有密码的

sudo passwd root，注意输入密码的时候可能看不见，实际上是有输入的

2：解锁root

sudo passwd –unlock root

3：锁定root

sudo passwd –unlock root

在SSH远程中使用root连接

编辑 /etc/ssh/sshd_config 中

sudo nano /etc/ssh/sshd_config

PermitRootLogin yes #值更改为yes

在FTP中使用root连接

编辑 /etc/ftpusers 中

sudo nano /etc/ftpusers

在root前边加#, 注释掉它 sudo service vsftpd restart 这样就可以用root用户登录了.
